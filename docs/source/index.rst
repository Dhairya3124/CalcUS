CalcUS - Computational Chemistry Web Platform 
=============================================

**CalcUS** aims to democratize access to computational chemistry by providing a user-friendly web interface to simplify running and analyzing quantum mechanical calculations. It is freely available, expandable and customizable. It promotes connectivity to multiple software packages and algorithms, thus providing state-of-the-art techniques to all practitioners.

For more information about the goal of the project, see the `preprint paper <https://chemrxiv.org/engage/chemrxiv/article-details/61b50d7502d90dc9845ec11f>`_.

.. note::

   This project is under active development.
